# Veo4AI GitHub Pages Site

This repository contains a lightweight static GitHub Pages site for Veo4AI.

The site is built with plain HTML, CSS, and a small amount of vanilla JavaScript. It has no backend, no login, no payment flow, no external dependencies, and no API keys.

Pages:

- `index.html`
- `veo-4-free.html`
- `veo-4-prompt-generator.html`

Veo4AI is an independent resource and is not affiliated with Google or Google DeepMind.
